About Me

Hey there! I'm Joanthan Louisa, a Full Stack Web Developer with a knack for turning ideas into pixel-perfect websites. I'm all about crafting digital magic that not only looks great but also works seamlessly.


I'm fluent in languages like HTML, CSS, JavaScript, and more. From creating stunning front-end interfaces to making the back-end run like a well-oiled machine, I've got it covered.
